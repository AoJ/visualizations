Nutrients in Food

An interactive visualization of the USDA Nutrient Database. Each food traces a line across the colorized parallel coordinates based on its nutrient content. The lines are colored by food group.

If you are unfamiliar with parallel coordinates, read this tutorial.

First, select some data...

  • Click and drag along an axis to create a selection
  • To move a selection, drag it up and down
  • To remove a selection, click the axis line behind it

Then...

  • Keep data you're interested in
  • Remove data you're not interested in, such as outliers

Refresh the page to start over.

When you're done with these instructions, hide them.
